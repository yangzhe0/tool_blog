本博客基于 [Fuwari](https://github.com/saicaca/fuwari) 模板构建，使用 [Astro](https://astro.build/) 框架，部署在 Vercel。

## 文章 Frontmatter

每篇文章顶部需包含以下字段：

```yaml
---
title: 文章标题
published: YYYY-MM-DD
description: '50字以内摘要'
image: './image/YYMMDD_01.jpg'
tags: [Tag1, Tag2]
category: Notes
draft: false
---
```

| 字段 | 说明 |
|------|------|
| `title` | 文章标题 |
| `published` | 发布日期，不加引号 |
| `description` | 摘要，用单引号包裹，无内容写 `''` |
| `image` | 封面图路径，无图写 `''` |
| `tags` | 最多 2 个，首字母大写英文 |
| `category` | 只能用：`Notes` / `Work` / `LOL` / `Life` |
| `draft` | `true` 则不显示在博客中 |

## 文件与目录规范

文章文件名格式：`YYMMDD_Title.md`，平铺在 `src/content/posts/` 根目录。

图片统一存放在 `src/content/posts/image/`，命名格式 `YYMMDD_01.png`，引用时用 `./image/YYMMDD_01.png`。

```
src/content/posts/
├── image/
│   ├── 250104_01.png
│   └── 260315_01.jpg
├── 250104_Sofa.md
├── 260315_PentaKill.md
└── ...
```

## 常用命令

| 命令 | 说明 |
|------|------|
| `pnpm dev --host` | 启动本地开发服务器（支持局域网访问） |
| `pnpm build` | 构建到 `./dist/`（Vercel 自动执行，本地一般不需要） |
| `pnpm preview` | 本地预览构建结果 |
| `pnpm astro check` | 检查类型错误 |
